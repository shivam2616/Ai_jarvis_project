# AI Jarvis Assistant

AI Jarvis is a voice-controlled virtual assistant built using Python.  
It can perform tasks like opening websites, telling time, searching Wikipedia, and more.

---

## Features

- 🎤 Voice Command Recognition
- 🌐 Open Websites (YouTube, Google, Facebook, etc.)
- 🕒 Tell Current Time
- 📖 Wikipedia Search
- 💬 Text-to-Speech Response

---

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3 (Text-to-Speech)
- pywhatkit
- Wikipedia API
- Webbrowser
